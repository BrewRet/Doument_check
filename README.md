# Document check

Демо-приложение для ШИФТ Интенсив.

Получилось реализовать все кроме тестов эндпооита с зависимостями и реализации сбора и мониторинга метрик.

## Быстрый старт

### 1. Установите зависимости

uv sync

### 2. Соберите и запустите докер контейнер

docker compose up --build

Сервер запустится на `http://localhost:8080`
### 3. Тесты запускаются локально

uv run pytest -v

### 4. Проверка работы запросов

Перейдите на `http://localhost:8080/docs`, здесь можно протестировать работу эндпоинта
